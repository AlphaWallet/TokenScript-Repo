Make Tokens Smart

Today, we have blockchain tokens. They are defined in Smart Contracts. They have values.

TokenScript allows token to do more.

  • For a token that represents ownership of a car, with TokenScript, you can start the engine.
  • For a token that represents a book, with the help of TokenScript, you can read it.

And all these are still following the rules in Smart Contracts.

The tokens that can do these is called Smart Tokens.

TokenScript let's you make Smart Tokens.
